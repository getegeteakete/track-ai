# MAI 配車AI管理システム — デモ

空車自動配車AI管理システムのインタラクティブデモアプリケーション。

## 機能

- **📊 ダッシュボード** — KPI、週間利益推移、サイト別実績
- **🚛 配車ボード** — 車両ステータス管理、案件マッチング、ワンタップ配車
- **💬 LINE Bot** — 4つの会話シナリオをリアルタイム再現
- **💰 利益分析** — サイト別・ルート別・車両別パフォーマンス

## 対応サイト

- 🟠 Amazon Relay（自動落札対応）
- 🔵 トラボックス（求荷求車）
- 🟢 WebKIT（組合ネットワーク）
- 🔴 JLローカルネット（九州ネットワーク）

## 開発

```bash
npm install
npm run dev
```
